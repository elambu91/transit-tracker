Flash Transit Tracker Firmware (64x32)

Hebrew Font Support + RTL Mode

Before You Start:

  • Use Google Chrome or Microsoft Edge browser
  • Connect your Transit Tracker to your computer via USB
  • This firmware includes Hebrew font support (ChavaRegular) and an RTL Mode switch
  • Compatible with 64x32 displays (2 panels, 128x32 total)

After Flashing:

  1. The device will reboot and create a WiFi access point
  2. Connect to the "transit-tracker" network
  3. Configure your WiFi credentials through the captive portal
  4. Access the device's web interface to configure routes
  5. Enable "RTL Mode" from the configuration if needed for Hebrew text

Features in This Build:

  • ✅ Hebrew character support (all Hebrew letters)
  • ✅ RTL Mode configuration switch (for future use)
  • ✅ All standard Transit Tracker features
  • ⚠️ Note: Full RTL text rendering requires component modifications (planned for future update)
